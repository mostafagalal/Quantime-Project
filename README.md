**Business Problem**

The objective of this project is to analyze customer purchasing behavior and sales performance across products, brands, and customer segments.
The analysis aims to support commercial decision making related to product prioritization, customer segmentation, and demand patterns over time.

**Data**

The dataset consists of transactional retail data, including:
- Transaction date and store identifier.
- Customer and loyalty card identifiers.
- Product details (brand, flavor, package size).
- Sales value and quantity purchased.
- Customer attributes (premium status, age stage, marital status, family size).
- Pack size categorization for consumption analysis.

**Key Columns**


**Approach**

- Date : Transaction date

- Store_Nbr : Store identifier

- Lylty_Card_Nbr : Customer loyalty card ID

- Txn_Id : Transaction ID

- Prod_Nbr : Product identifier

- Prod_Qty : Quantity purchased

- Tot_Sales : Total sales value

- Package_Size : Product pack size

- Brand / Flavor  : Product attributes

- Premium_Customer : Customer premium classification

- Age_Stage : Customer age group

- Marital / Fam : Household and family indicators

- Package_bins : Pack size grouping for analysis
  
**Tools**

- SQL
- Python
- Power BI (for visualization and insight presentation)

**Key Insights**

- Identified variations in demand across brands and package sizes, indicating different consumption behaviors by customer segment.
- Observed differences in sales contribution between Package Sizes.
- Highlighted customer age and family segments associated with higher purchase volumes and total sales.
- Found out that Kettel & Chicken is the most selling product & flavour.

**Decision Supported**

The analysis highlights key demand drivers across products, customer segments, and pack sizes, providing commercial insight into product prioritization, customer segmentation, and sales performance evaluation.
